# Library app
Backend: Express, Sequelize
Frontend: React, Redux and Immutable.js

### Install:
```
npm install && cd src/client && npm install
```

### Run app in development mode:
```
npm run dev
```

### Build:
```
cd src/client && npm run build
```

### Start app:
```
npm start
```
